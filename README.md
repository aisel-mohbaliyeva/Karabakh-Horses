# 🐎 Karabakh Horses

## 📌 Overview
Karabakh Horses is a modern front-end project showcasing the beauty, elegance, and strength of Karabakh horses — one of Azerbaijan’s national treasures.

This project demonstrates my first practical experience in building a responsive and clean website.

---

## 🎯 Goal
The goal of this project is to strengthen my front-end development skills and expand it further in the future with more features, animations, and interactive elements.

---

## 🛠 Technologies Used
- HTML  
- CSS  
- JavaScript

## 🚀 Features
- Responsive layout for desktop and mobile  
- Clean modern design
